사용 방법

📋 기본 사용법

  1. 앱 추가: "새 앱 추가" 버튼을 클릭하여 새로운 앱을 등록합니다.
  2. 앱 수정: 각 행의 "수정" 버튼을 클릭하여 앱 정보를 변경합니다.
  3. 앱 삭제: "삭제" 버튼으로 앱을 제거합니다.
  4. 표시/숨김: 체크박스를 클릭하여 홈페이지에 표시할지 결정합니다.
  5. 순서 변경: 행을 드래그하여 앱의 순서를 조정합니다.

📁 카테고리 관리

  • "카테고리 관리" 버튼으로 카테고리를 추가/수정/삭제할 수 있습니다.
  • 카테고리를 삭제하면 해당 카테고리의 앱들은 자동으로 삭제됩니다.

💾 저장 방법

  1. 모든 수정을 완료한 후 "JSON 다운로드" 버튼을 클릭합니다.
  2. 다운로드된 apps.json 파일을 프로젝트 폴더에 저장합니다 (기존 파일 덮어쓰기).
  3. Git으로 커밋하고 GitHub에 푸시합니다: 
    git add apps.json
git commit -m "앱 목록 업데이트"
git push
  4. 몇 분 후 GitHub Pages에 자동으로 반영됩니다!

📤 JSON 업로드

기존 apps.json 파일을 업로드하여 계속 편집할 수 있습니다.
